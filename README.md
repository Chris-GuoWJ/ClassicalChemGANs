# ClassicalChemGANs
A set of GANs model which is used to predict new energy data from random noise.
CCGANs1 is the most accurate model.
CCGANs2 is a simplified model of CCGANs1, which provides a faster computing speed but not accurate as CCGANs1.
CCGANs3 uses a different frame from CCGANs1 and CCGANs2, which inherits the frame of NN model in my another repository.
